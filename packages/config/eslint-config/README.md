# @tne-kit/eslint-config
## Eslint Config
Shared Eslint config for @way project and micro/monolithic frontend projects.

## Installation

### npm
```sh
npm install @tne-kit/eslint-config
```
### yarn
```sh
yarn add @tne-kit/eslint-config
```
## How to use
Add to eslint.js:
```sh
extends: ["@tne-kit/eslint-config"],
```
