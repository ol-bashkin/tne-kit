# @tne-kit/styles
## Styles library
Styles library include:
- Grid system
- Color system
- Typography
- Misc styles

## Installation

### npm
```sh
npm install @tne-kit/styles
```
### yarn
```sh
yarn add @tne-kit/styles
```
## How to use
Add to your main.*:
```sh
import "@tne-kit/styles";
```
